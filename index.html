Open Cellular Vehicle To Everything (OpenCV2X) - Mode 4

A 3GPP compliant CV2X Mode 4 Open Source implementation for OMNeT++

An open source implementation of the 3GPP standard CV2X in Mode 4. It is based on an extended version of the SimuLTE OMNeT++ simulator which enables LTE network simulations. It also integrates with the Artery framework, which provides ITS-G5 standardisation.

If you use our CV2X model, we would appreciate a citation of our work.

Brian McCarthy and Aisling O'Driscoll,
"OpenCV2X Mode 4: A Simulation Extension for Cellular Vehicular Communication Networks,
Submitted to "1st ACM Workshop on Technologies, mOdels, and Protocols for Cooperative Connected Cars (TOP-Cars), Co-located with ACM MobiHoc (TOP-Cars@MobiHoc), Italy, July 2019.
