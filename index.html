Open Cellular Vechicle To Everything (C-V2X)

A 3GPP compliant Open Source C-V2X implementation for OMNeT++

Our open source implementation of the 3GPP standard C-V2X. It is based on both the SimuLTE OMNeT++ simulator which allows for LTE network simulations, it also integrates with the Artery framework, which allows for our model to interact with the current ITS-G5 standard.

If you use our C-V2X model, we would appreciate a citation of our work.


Brian McCarthy and Aisling O'Driscoll
"OMNeT++ C-V2X Mode 4: An Extension for Cellular Vehicular Communication Networks,
"1st ACM Workshop on Technologies, mOdels, and Protocols for Cooperative Connected Cars (TOP-Cars)
