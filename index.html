Open Cellular Vehicle To Everything (OpenCV2X) - Mode 4

A 3GPP compliant CV2X Mode 4 Open Source implementation for OMNeT++







An open source implementation of the 3GPP standard CV2X (Rel 14) Mode 4. It is based on an extended version of the SimuLTE [link] OMNeT++ simulator which enables LTE network simulations.

Two variants are available. The first integrates with the Artery framework to provide full ITS-G5 standardisation across the entire communication stack. The second integrates with Veins [link] only.

If you use either CV2X model, we would appreciate a citation of our work.

Brian McCarthy and Aisling O'Driscoll,
"OpenCV2X Mode 4: A Simulation Extension for Cellular Vehicular Communication Networks",
IEEE International Workshop on Computer Aided Modeling and Design of Communication Links and Networks, Cyprus, September 2019.
