Open Cellular Vechicle To Everything (C-V2X)

A 3GPP compliant Open Source C-V2X implementation for OMNeT++

Features


3GPP standard C-V2X

Model is developed to the 3GPP standard for Cellular Vehicle to Everything communication.

Implementing the LTE Mode 4 sidelink communication through extension of the SimuLTE network simulator.


ITS-G5 Integration

Integrates Artery an ITS-G5 compliant implementation.

This integration allows for the investigation of C-V2X with the ETSI standards for vehicular communication.


Open Source

Allowing extension and reuse for the vehicular community.
