Frequently Asked Questions

Where can I get help

Help is available, though it is important to determine what help you need.

  • OMNeT++ issues should be directed to the OMNeT++ Google Group where people can help with issues with Network simulation
  • Issues with SUMO should be directed to the SUMO mailing list
  • Issues with the model can be directed to the GitHub Issues page and opening an issue
  • Otherwise StackOverflow is always a good resource.
    • https://stackoverflow.com/
Can I contribute

Contributions are most welcome, if contributing feel free to contact myself b.mccarthy@cs.ucc.ie with your suggestion. If you wish to contribute then please fork from the repo and then after development make a pull request on the original.
